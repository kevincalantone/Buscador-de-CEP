🔍 Buscador de CEP - React + Vite
Aplicação moderna desenvolvida em React para consulta de endereços brasileiros de forma simplificada, utilizando a API ViaCEP.

💻 Sobre o Projeto
Este projeto permite que o usuário informe um CEP e obtenha, em tempo real, informações detalhadas como logradouro, bairro, cidade, UF e até a região do país. Foi desenvolvido com foco no aprendizado de consumo de APIs, Hooks e renderização condicional no React.

✨ Funcionalidades
Busca em Tempo Real: Integração assíncrona com a API ViaCEP utilizando Axios.

Interface Responsiva: Design adaptável para qualquer tamanho de tela (Mobile/Desktop).

Feedback ao Usuário: Validação de campos vazios e tratamento de erros para CEPs inexistentes.

Estilização Moderna: Uso de animações CSS para exibição dos resultados.

🚀 Tecnologias Utilizadas
React.js (Vite)

Axios (Requisições HTTP)

React Icons (Ícones Feather)

CSS3 (Flexbox e Animations)
